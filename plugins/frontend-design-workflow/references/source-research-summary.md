# Source Research Summary

This bundle was seeded from `frontend_design_research_lowcode_datascience.md` and the provided six-layer design framework image, then generalized for broad frontend, UI/UX, web, desktop, and cross-platform app design. Current standards notes were checked against public sources in May 2026.

## Current Standards Notes

- WCAG 2.2 became a W3C Recommendation on 2023-10-05 and is the default modern target for public web accessibility planning.
- The European Accessibility Act entered application on 2025-06-28 for covered EU products and services.
- Interaction to Next Paint replaced First Input Delay as a Core Web Vital on 2024-03-12; Core Web Vitals are LCP, INP, and CLS.
- The W3C Design Tokens Community Group standardizes token exchange format work; treat design token specs as community-group standards rather than W3C Recommendations unless a newer official status is verified.
