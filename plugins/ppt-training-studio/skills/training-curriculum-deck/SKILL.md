# Training Curriculum Deck

Use this skill for PowerPoint decks meant to teach, train, onboard, or facilitate practice.

Read `../../references/training-deck-design.md` for substantial work. Read `../../references/powerpoint-production-handoff.md` before creating a PPTX.

## Workflow

1. Define learner profile, prerequisites, environment, timebox, and success outcome.
2. Convert topics into measurable learning objectives.
3. Chunk the deck into short learning segments.
4. For each segment, plan: explain, show, practice, check, summarize.
5. Choose visuals that teach: diagrams, examples, screenshots, flows, contrast pairs, or short videos.
6. Add facilitator notes, timing, prompts, and answer keys when useful.
7. If producing `.pptx`, invoke the `Presentations` skill once after this strategy is set.

## Reject

- Slide-after-slide lecture bullets.
- Decorative imagery that does not teach.
- Activity slides without instructions or success criteria.
- Knowledge checks without feedback.
- Dense text that should be in notes or handouts.
