# Training Activity Slides

Use this skill when a training deck needs active learning, practice, or assessment slides.

Read `../../references/training-deck-design.md` when planning a full module.

## Activity Workflow

1. State the learning objective.
2. Choose activity type: recall, classify, debug, compare, role-play, scenario, lab, reflection, or teach-back.
3. Give clear instructions, timebox, materials, and success criteria.
4. Include answer reveal, debrief prompt, or worked solution.
5. Use icons or color bands to mark mode: learn, demo, practice, check, discuss.
6. Keep activity slides visually simple and instructionally complete.

## PowerPoint Components

- Timer or timebox label.
- Prompt and constraints.
- Workspace area or screenshot.
- Answer/reveal slide.
- Facilitator note with expected responses.
- Optional video/demo link with fallback still.

## Gate

A learner should know exactly what to do without asking the facilitator to decode the slide.
